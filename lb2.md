Реалізувати позитивний вплив наявності на імітаційному полі електростанцій на вірогідність появи нових людей. Збільшити вірогідність висадки дерев у клітинах поблизу електростанцій.

### **Внесені зміни у вихідну логіку моделі:**

### Зміна 1

Для реалізації цього ефекту було змінено значення ймовірності народжуваності на локальному рівні. 

Початковий код:

```
to reproduce  ;; person procedure
  if health > 4 and random-float 1 < birth-rate [
    hatch-people 1 [
      set health 5
    ]
  ]
end
```

Базове значення параметра збільшується у **1.5 раза**, якщо людина знаходиться на клітині з електростанцією або в радіусі 2 клітин від неї.

Допрацьований код:

```
to reproduce  ;; person procedure
  let local-birth-rate birth-rate

  ;; якщо поряд є електростанція — збільшуємо ймовірність народження
  if any? patches in-radius 2 with [ is-power-plant? ] [
    set local-birth-rate birth-rate * 1.5
  ]

  if health > 4 and random-float 1 < local-birth-rate [
    hatch-people 1 [
      set health 5
    ]
  ]
end
```

### Зміна 2

### Збільшення ймовірності висадки дерев поблизу електростанцій

Люди активніше висаджують дерева біля джерел забруднення.

**Підвищення ймовірності висадки дерев поблизу електростанцій**

Аналогічно, для процесу висадки дерев було змінено локальне значення параметра `planting-rate`. У разі, якщо поблизу (в радіусі 2 клітин) є електростанція, ймовірність посадки дерев збільшується у **2 рази** відносно базового значення:

`local-planting-rate = planting-rate × 2`.

Таким чином, у моделі були змінені саме **коефіцієнти впливу електростанцій** на процеси народжуваності та висадки дерев, що дозволяє кількісно оцінювати їхній позитивний локальний ефект.

Код до зміни:

```
to maybe-plant  ;; person procedure
  if random-float 1 < planting-rate [
    hatch-trees 1 [
      set health 5
      set color green
    ]
  ]
end
```

Код після зміни:

```
to maybe-plant  ;; person procedure
  let local-planting-rate planting-rate

  ;; якщо поряд електростанція — збільшуємо ймовірність посадки дерев
  if any? patches in-radius 2 with [ is-power-plant? ] [
    set local-planting-rate planting-rate * 2
  ]

  if random-float 1 < local-planting-rate [
    hatch-trees 1 [
      set health 5
      set color green
    ]
  ]
end
```

Зміна 3 (на власний розсуд)

## Негативний вплив високого рівня забруднення на народжуваність

У модель було додано механізм, який враховує негативний вплив надмірного забруднення навколишнього середовища на процес відтворення населення. Якщо людина перебуває в зоні з високим рівнем забруднення, ймовірність появи нових людей зменшується, навіть за наявності позитивних факторів, таких як близькість до електростанцій. Це відображає погіршення умов життя та здоров’я населення в екологічно несприятливих регіонах і робить модель більш реалістичною.

### Змінені значення

**Поріг забруднення:** `pollution > 2`

**Коефіцієнт зменшення народжуваності:** `0.5`

Формула: `local-birth-rate = local-birth-rate × 0.5`

Код реалізації:

```
to reproduce  ;; person procedure
  let local-birth-rate birth-rate

  ;; позитивний вплив електростанцій
  if any? patches in-radius 2 with [ is-power-plant? ] [
    set local-birth-rate birth-rate * 1.5
  ]

  ;; негативний вплив високого забруднення
  if pollution > 2 [
    set local-birth-rate local-birth-rate * 0.5
  ]

  if health > 4 and random-float 1 < local-birth-rate [
    hatch-people 1 [
      set health 5
    ]
  ]
end
```

Ще у модель було додано механізм поетапного впливу забруднення на здоров’я населення. При незначному рівні забруднення здоров’я погіршується повільно, при середньому – швидше, а при критично високому забрудненні відбувається різке зниження рівня здоров’я людей. Такий пороговий характер впливу відображає токсичність шкідливих речовин у довкіллі та дає змогу змоделювати накопичення екологічних ризиків для населення.

Код реалізації

```jsx
to eat-pollution  ;; person procedure

  ;; низький рівень забруднення – слабкий вплив
  if pollution > 0.5 and pollution <= 2 [
    set health health - (pollution / 12)
  ]

  ;; середній рівень – помітне погіршення здоров'я
  if pollution > 2 and pollution <= 4 [
    set health health - (pollution / 8)
  ]

  ;; критичне забруднення – різке зниження здоров'я
  if pollution > 4 [
    set health health - 0.5
  ]

  ;; не дозволяємо здоров'ю стати нижче нуля
  if health < 0 [
    set health 0
  ]
end
```

### Висновки

Проведені експерименти показали, що баланс між людською діяльністю та станом довкілля є ключовим для стабільності системи. Початкова версія моделі мала обмежену стійкість до зростання забруднення.

Після внесення змін модель стала більш реалістичною та стабільною: посилився позитивний ефект висадки дерев, з’явився компенсувальний вплив електростанцій і обмеження народжуваності за високого забруднення. Порівняння результатів до і після змін підтвердило зростання тривалості існування системи, що підкреслює важливість поєднання технологічного розвитку з екологічними заходами.
